Creative Archive

Photoshop composites and promotional art for bands, theater, and clients — created under BlackIce Multimedia.

Add your photos

Export images from Apple Photos into this project, then list them in the works file.

  1. Open Photos on your Mac and select the pieces you want.
  2. Choose File → Export → Export Photos…
  3. Use JPEG, and set size to Large or about 2000px on the long edge.
  4. Save them into assets/images/archive/ in this repo.
  5. Edit assets/images/archive/works.json so it looks like this:
[
  {
    "file": "band-poster.jpg",
    "title": "Band Poster",
    "caption": "Promotional art for a local band"
  },
  {
    "file": "play-flyer.jpg",
    "title": "Theater Flyer",
    "caption": "Poster design for a stage production"
  }
]

No Photoshop license needed — export from Photos is enough. Use the exact file names you saved.
